Die MAC-Adresse ist eine eindeutige Identifizierung für jede Netzwerkkarte. Je nach installiertem Betriebssystem wird diese Adresse an anderen Stellen angezeigt.
Linux:
/sbin/ifconfig aufrufen, z.B:
[root@blackbox /]# ifconfig eth0
eth0     Link encap:Ethernet  HWaddr 00:10:00:E7:7E:E7

Windows 2000/XP:
Start -> Ausführen -> "cmd" eingeben -> "ipconfig /all" aufrufen, z.B:
Microsoft Windows XP [Version 5.1.2600]
(C) Copyright 1985-2001 Microsoft Corp.

C:\>ipconfig /all
