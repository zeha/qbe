Die Internetzugangskontrolle ist mehrstufig, nach Klassen und Gruppen, aufgebaut.

Zuerst sehen Sie die Klassenliste, wobei rechts neben jedem Klassennamen der aktuelle Status angezeigt wird. Zum Beispiel:

1AHET:
 
Bedeutung: Internet für die gesamte Klasse gesperrt.


1AHET:
 
Bedeutung: Internet für die gesamte Klasse freigegeben.


1AHET:
 
Bedeutung: Es wurde für einzelne Gruppen die Internetfreischaltung geändert.
