\subsection{Abstrakt}

Dieses Projekt entstand aus der schier unmöglichen Aufgabe, 1500 Schülern den Zugriff auf ein neues HTL-Netzwerk sowie das Internet zu geben -- mit möglichst wenig Geld, und keinem zusätzlichen Verwaltungsaufwand. Dass dies so nicht funktionieren würde, war von Anfang an klar, jedoch konnte man immer noch versuchen den neuen Verwaltungsaufwand möglichst gering zu halten. Damit war das Ziel klar: neue Systeme mit neuer Software mussten beschafft werden, um diese 1500 Benutzer zentral so verwaltbar zu machen, dass dies eine Person neben der normalen Lehrverpflichtung schaffen kann.

Während der 3 Jahre des Projektaufbaus kamen immer neue Anforderungen hinzu, neue Systeme (Informationssystem, Wetterstation, Inventarverwaltung) mussten eingebunden werden.
